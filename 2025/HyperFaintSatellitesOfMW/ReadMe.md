This folder contains parameter files for the paper:

* Title: _"The abundance and properties of the lowest luminosity dwarf galaxies around the Milky Way: Insights from Semi-Analytic Models"_
* Author(s): Ahvazi et al.
* Publication year: 2025
* URLs:
  * [arXiv](https://arxiv.org/abs/2511.15808)
  
* Galacticus commit hash: [ee197e314ac28f1eca3b6dcc47c5f88682bbddb5](https://github.com/galacticusorg/galacticus/commit/ee197e314ac28f1eca3b6dcc47c5f88682bbddb5)
* Datasets commit hash: [6c7685d6150ffa81b5726bd9a9555a45a911f513](https://github.com/galacticusorg/datasets/commit/6c7685d6150ffa81b5726bd9a9555a45a911f513)

This directory contains the Galacticus parameter `.xml` files used to produce the fiducial and No-H<sub>2</sub> model datasets presented in the paper.
